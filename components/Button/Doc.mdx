### Twig

#### Primary Button

```twig
<button class="t-body-small t-interactive-sub button--primary c-content-interactive-primary c-bg-interactive-primary ani-background">{{ text }}</button>
```

#### Primary Button - Disabled

```twig
<button class="t-body-small t-interactive-sub button--primary c-content-interactive-primary c-bg-interactive-primary ani-background button--disabled">{{ text }}</button>
```

```twig

  {{ text }}
</a>
```
