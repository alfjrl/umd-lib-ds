### Features

There are only 2 types of button will be used in the design system, which are:

- Primary
- Secondary

The primary button serves as the main action button, while the secondary button is used for "link to more" functions.

### Twig

#### Primary Button

```twig
<button class="t-body-small t-interactive-sub button--primary c-content-interactive-primary c-bg-interactive-primary ani-background">{{ text }}</button>
```

#### Primary Button - Disabled

```twig
<button class="t-body-small t-interactive-sub button--primary c-content-interactive-primary c-bg-interactive-primary ani-background button--disabled">{{ text }}</button>
```

#### Secondary Button

```twig
<a class="t-body-small t-interactive-sub button--secondary c-content-primary c-underline-primary ani-underline">
  <div class="i-chevron"></div>
  {{ text }}
</a>
```

#### Secondary Button - Disabled

```twig
<a class="t-body-small t-interactive-sub button--secondary c-content-primary c-underline-primary ani-underline button--disabled">
  <div class="i-chevron"></div>
  {{ text }}
</a>
```
